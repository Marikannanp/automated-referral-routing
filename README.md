# Automated Referral Lead Routing Engine

## 📌 Problem (Situation)
Manual operational routing of incoming sales references caused communication delays. 
Because field listings and customer classifications were manually sorted, data latency 
led to internal tracking mismatches and slower response times.

## 🛠️ Solution & Action
Engineered an automated script using JavaScript/Google Apps Script to capture data objects directly from web forms/Google form. 
- Designed conditional code architecture (`if/else`) to instantly distinguish between Existing Customers looking for a second property and New Referral leads.
- Configured dynamic HTML string layouts to output cleanly formatted, bolded internal email updates.
- Programmed an asynchronous multi-tier CC list structure to auto-loop assigned executives and line management via the Gmail API instantly upon form submission.

## 📈 Business Impact
- Accelerated data hand-off speeds to under <1 second per submission.
- Prevented team pipeline classification errors.
- Secured full operational transparency for high-priority property bookings.

## 💻 Tech Stack Used
- JavaScript (Google Apps Script Engine)
- Google Forms & Sheets API
- Gmail Communication Protocol / SMTP
